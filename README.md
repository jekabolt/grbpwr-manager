# grbpwr-manager

service for upload and manage stock products [grbpwr.com](https://grbpwr.com)

# How to run

#### for local development

```shell script
    make run
```

#### Build docker image

```shell script
    make image
```

#### Run in docker

```shell script
    make image-run
```
